# Task Plan

Active task plans go here. Use checkable items.

## Template

```
## Task: <name>

### Goal
<what we're building/fixing>

### Plan
- [ ] Step 1
- [ ] Step 2
- [ ] Step 3

### Verification
- [ ] npm run typecheck passes
- [ ] npm run lint:check passes
- [ ] npm run test passes
- [ ] Manual checklist: <describe manual steps>

### Review
<Add notes after completion: what changed, any surprises>
```
